# מאסטר הפנייה | Penia Master

משחק לימודי עצמאי לתרגול **פנייה** (פריטה יוונית) על בוזוקי — מושלם לשיתוף בקבוצות, שיעורים ווואטסאפ.

**קישור חי (Vercel):** אחרי חיבור ל-GitHub, Vercel ייתן כתובת כמו `https://penia-master.vercel.app`

**אקדמיה מלאה:** [בוזוקי אקדמי](https://boaz65sa-byte.github.io/bouzoukifret/)

## הפעלה מקומית

פתחו `index.html` בדפדפן (Chrome / Edge / Safari).

## פריסה ב-Vercel (מומלץ לקבוצות)

1. העלו את הריפו הזה ל-GitHub (ריפו נפרד — רק המשחק)
2. היכנסו ל-[vercel.com](https://vercel.com) → **Add New Project**
3. בחרו את הריפו `penia-master`
4. **Framework Preset:** Other (אין build — אתר סטטי)
5. **Deploy** — תוך דקה תקבלו קישור לשיתוף בוואטסאפ

אין צורך ב-`npm install` או ב-build command.

## מה יש במשחק

- **8 שלבים מדורגים** — מרבעים למטה ועד טזנה 7/8
- **משחק קצב דו-כיווני** — חיצים ↓↑, ניקוד, קומבו, כוכבים
- **פרופילי שחקנים** — כל נגן בקבוצה עם שם ואווטאר
- **לוח תוצאות** — התחרות בין הנגנים
- **שיתוף ציון** — כפתור "שתף בקבוצה" לוואטסאפ
- **סטטיסטיקות קהילה** (אופציונלי) — Firebase

## שליטה

| מכשיר | ↓ למטה | ↑ למעלה |
|--------|--------|--------|
| מקלדת | ↓ או J | ↑ או K |
| מגע | חצי תחתון של המסלול | חצי עליון |

## סטטיסטיקות קהילה (Firebase — חינם)

1. צרו פרויקט ב-[Firebase Console](https://console.firebase.google.com)
2. הפעילו **Realtime Database** (במצב test לפיתוח)
3. העתיקו `config.example.js` → `config.js` והדביקו את הפרטים
4. במסך הבית יופיעו: **כמה נגנים** ו**כמה משחקים** בכל הקבוצות

> `config.js` לא עולה ל-Git — הגדירו אותו ישירות ב-Vercel Environment Variables או העלו מקומית בלבד.

### כללי אבטחה מומלצים (Firebase Rules):

```json
{
  "rules": {
    "stats": {
      ".read": true,
      "players": { ".write": true },
      "games": { ".write": true }
    }
  }
}
```

## שיווק לקבוצות בוזוקי

- שלחו קישור + QR code לקבוצת הוואטסאפ
- בכל שיעור: כל נגן יוצר פרופיל → משחק שלב → לוח תוצאות
- "שתף בקבוצה" מעתיק טקסט מוכן לוואטסאפ
